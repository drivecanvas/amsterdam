The followings CSS rules are intended to apply a certain width or height to a DOM element.

Proportion and Fixed sizings

The following classes are availables:

  • .width-* { width: *px; }
  • .height-* { height: *px; }
  • .min-height-* { min-width: *px; }
  • .max-height-* { max-width: *px; }

The wildcard (*) refers to a specific measure, follow this guide to use any of the following measure. Proportion based or fixed sizing.
